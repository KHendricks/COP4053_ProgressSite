This project is developed in the Unity Game Engine by James, Faye, and Kyle. This site will document the weekly progress of the game for those involved in the development process.

Week One : 24 - August - 2018

For the first week of the project James, Faye and I agreed to be on a team and work in the Unity Game Engine for the project. We discussed potential game ideas ranging from old school Mario games to personal ideas such a cowboy style of game with 2D graphics portrayed in 3D. In addition to discussing potential game types, we agreed to use GitHub for version control and have the repository for the project here. Kyle created a prototype for a main menu to get the ball rolling for the project and to provide a foundation for the group to work on. Lastly, the repository for the website is hosted on GitHub here. By using GitHub to host the website, it allows for a timeline display to see how frequently updates were pushed to the website for verification of the progress.

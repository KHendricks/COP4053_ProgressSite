COP4053 - Team 2

This project is developed in the Unity Game Engine by James, Faye, and Kyle. This site will document the weekly progress of the game for those involved in the development process. Each weekly section will have a paragraph and either a video or picture showcasing the progress made during that week. In addition to that, at the bottom of the page is an image gallery with any additional images and a description about the image.

  • Get Started

Week One : 24 - August - 2018

For the first week of the class James, Faye and Kyle agreed to be on a team and work in the Unity Game Engine for the project. We discussed potential game ideas ranging from old school Mario games to personal ideas such a cowboy style of game with 2D graphics portrayed in 3D. In addition to discussing potential game types, we agreed to use GitHub for version control and have the repository for the project here. Kyle created a prototype for a main menu to get the ball rolling for the project and to provide a foundation for the group to work on. Lastly, the repository for the website is hosted on GitHub here. By using GitHub to host the website, it allows for a timeline display to see how frequently updates were pushed to the website for verification of the progress.

Week Two : 31-August-2018

This week we all got together to solidify the idea of what we wanted to work on for the project and began heading in the desired direction. We decided to pursue Faye’s idea of a Western style of game with a setting and story revolving around that theme. The playstyle will be something of a mixture of 2D and 3D allowing the player to experience the game with a different approach because of the art style. We delegated roles for the project in which Faye will handle the graphics and animation side of the game, while James and Kyle will handle setting up the basic mechanics for the game for the time being. We all met up on Wednesday to work on the Project Proposal presentation to knock that out of the way. James pulled in some 3D assets fitting the theme of the game into the Unity project and Kyle set up a basic level using those assets with a simple “player” with WASD movement controls.
